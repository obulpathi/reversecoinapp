Code

As a proof of concept, we created Reversecoin based on Bitcoinpy, first python implementation of Bitcoin.

> git clone git@github.com:obulpathi/vaultcoin.git

Get the source code at Github.

Installation

Install Reversecoin from PyPI, using pip:

> pip install reversecoin

Checkout the documentation, to see how it works.

Configuration

Look to ~/.bitocoinpy.cfg for configuraiton options.

> vim ~/.bitocoinpy.cfg

Testing

We use tox to do automated testing. Give it a try with:

> tox -e py27

Contribute

You are welcome to submit patches to Reversecoin.

> git commit -m "my awesome commit"
